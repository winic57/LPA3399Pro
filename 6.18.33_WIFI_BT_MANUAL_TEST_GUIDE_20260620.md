# RK3399Pro 6.18.33 WiFi / Bluetooth 手动测试指南

生成时间：2026-06-20 17:29:10 CST

依据：`/mnt/sdb3/LPA3399Pro/6.18.33_GMAC_PATCH_COMPILE_VERIFY_20260618.md`

## 0. 当前结论

板载无线芯片为 **RTL8821CS**：

- WiFi 走 SDIO，使用 mainline `rtw88` 驱动：`rtw88_8821cs`
- Bluetooth 走 UART，使用：`hci_uart + btrtl`
- 预期接口：
  - WiFi：`wlan0`
  - Bluetooth：`hci0`

注意：不要再按早期 Broadcom 方案测试 `brcmfmac`。

---

## 1. 基础状态检查

在板端 TTL/SSH shell 执行：

```bash
uname -r
ip -br link
rfkill list
lsmod | grep -E 'rtw88|mac80211|cfg80211|bluetooth|hci_uart|btrtl|btbcm|btqca'
dmesg | grep -iE 'rtw88|rtl|wlan|wifi|bluetooth|hci|firmware' | tail -120
```

预期能看到：

```text
wlan0
hci0
rtw88_8821cs
rtw88_8821c
rtw88_sdio
rtw88_core
mac80211
cfg80211
hci_uart
btrtl
bluetooth
```

# 2. 手动测试 WiFi

## 2.1 加载 WiFi 驱动

```bash
modprobe rtw88_8821cs
rfkill unblock wifi
ip link set wlan0 up
```

检查：

```bash
ip -br link show wlan0
iw dev
rfkill list
```

预期：

```text
wlan0 UP
phy0: Wireless LAN
Soft blocked: no
Hard blocked: no
```

## 2.2 扫描 AP

```bash
iw dev wlan0 scan | grep -E 'SSID:|freq:|signal:'
```

或者用 NetworkManager：

```bash
nmcli dev wifi rescan ifname wlan0
nmcli dev wifi list ifname wlan0
```

预期能看到 2.4GHz / 5GHz AP。

## 2.3 连接 WiFi

替换成实际 SSID 和密码：

```bash
nmcli dev wifi connect "你的SSID" password "你的密码" ifname wlan0
```

检查连接和地址：

```bash
ip -br addr show wlan0
iw dev wlan0 link
nmcli dev show wlan0 | grep -E 'IP4.ADDRESS|IP4.GATEWAY|GENERAL.STATE'
```

预期：

```text
wlan0 UP 192.168.x.x/24
Connected to xx:xx:xx:xx:xx:xx
```

## 2.4 Ping 测试

按实际网关替换 `192.168.50.1`：

```bash
ping -I wlan0 -c 30 192.168.50.1
ping -I wlan0 -c 30 8.8.8.8
```

长测：

```bash
ping -I wlan0 -c 300 192.168.50.1
```

通过标准：

```text
0% packet loss
wlan0 不掉线
dmesg 无 rtw88 firmware crash / timeout
```

## 2.5 iperf3 吞吐测试

在局域网另一台机器上启动 server：

```bash
iperf3 -s
```

板端执行，替换 server IP：

```bash
WLAN_IP=$(ip -4 -o addr show wlan0 | awk '{print $4}' | cut -d/ -f1)
iperf3 -c 192.168.50.12 -B "$WLAN_IP" -t 300 -i 10
```

反向测试：

```bash
WLAN_IP=$(ip -4 -o addr show wlan0 | awk '{print $4}' | cut -d/ -f1)
iperf3 -c 192.168.50.12 -B "$WLAN_IP" -R -t 300 -i 10
```


## 3.2 打开蓝牙控制器

进入交互模式：

```bash
bluetoothctl
```

在 `bluetoothctl` 内执行：

```text
power on
agent on
default-agent
show
```

预期：

```text
Powered: yes
Pairable: yes
```

## 3.3 扫描蓝牙设备

在 `bluetoothctl` 内执行：

```text
scan on
```

等待 20~60 秒，应该看到附近设备：

```text
[NEW] Device XX:XX:XX:XX:XX:XX device-name
```

停止扫描：

```text
scan off
```

## 3.4 配对、信任、连接

替换成扫描到的 MAC：

```text
pair XX:XX:XX:XX:XX:XX
trust XX:XX:XX:XX:XX:XX
connect XX:XX:XX:XX:XX:XX
info XX:XX:XX:XX:XX:XX
```

通过标准：

```text
Pairing successful
Connection successful
Trusted: yes
Connected: yes
```

退出：

```text
quit
```
